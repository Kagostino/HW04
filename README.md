Q:週資料或月資料結果評比相似嗎 ?
A:週資料和月資料內容相近,中間分佈有些許差異外大致相同。
Q:不同指標評比結果相似嗎?
A:同指標對於ETF的衡量，著重於不同面向的表現，所以評斷出的結果並不是具有的相當類似的結果
